Welcome to ArduRPC
==================

ArduRPC brings `remote procedure calls(RPC) <http://en.wikipedia.org/wiki/Remote_procedure_call>`_ to microcontrollers. It has been developed for `Arduino <http://arduino.cc/>`_ based projects but can also be used with other platforms. The protocol has been designed to be simple and flexible.
